# NeuronIP Mobile Apps

This directory contains mobile application implementations for NeuronIP.

## Structure

- `ios/` - iOS application (Swift/Objective-C)
- `android/` - Android application (Kotlin/Java)

## Features

### iOS App
- Native iOS app built with SwiftUI
- Supports iOS 15.0+
- Core features:
  - Semantic search
  - Warehouse Q&A
  - Support tickets
  - Compliance dashboards
  - Workflow management

### Android App
- Native Android app built with Kotlin
- Supports Android API 24+
- Core features:
  - Semantic search
  - Warehouse Q&A
  - Support tickets
  - Compliance dashboards
  - Workflow management

## API Integration

Both mobile apps integrate with the NeuronIP REST API:
- Base URL: Configurable via app settings
- Authentication: API key or OAuth 2.0
- API version: v1
