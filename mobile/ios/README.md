# NeuronIP iOS App

Native iOS application for NeuronIP built with SwiftUI.

## Requirements

- iOS 15.0+
- Xcode 14.0+
- Swift 5.7+

## Features

- Semantic search
- Warehouse Q&A
- Support ticket management
- Compliance dashboards
- Workflow management
- Push notifications
- Offline support
