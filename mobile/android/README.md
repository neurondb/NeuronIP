# NeuronIP Android App

Native Android application for NeuronIP built with Kotlin.

## Requirements

- Android API 24+ (Android 7.0+)
- Kotlin 1.8+
- Android Studio Arctic Fox+

## Features

- Semantic search
- Warehouse Q&A
- Support ticket management
- Compliance dashboards
- Workflow management
- Push notifications
- Offline support
